Mohegan Sun Pocono

Company: Ipro Games

Duration: July 2022 – March 2024

Project Overview

Mohegan Sun Pocono is a social casino game offering players an immersive gaming experience with various casino-style games, including slots, poker, blackjack, and social game tournaments where players can win virtual chips. With more than 250+ social games from 5+ game providers, the platform ensures engaging and competitive gameplay through in-game purchases and virtual rewards.

Role & Responsibilities

  • ✅ Conducted manual and exploratory testing to identify and report critical bugs.
  • ✅ Configured daily push notifications to enhance player engagement and monitored bonus performance.
  • ✅ Designed and executed test cases while maintaining detailed documentation in JIRA.
  • ✅ Led User Acceptance Testing (UAT) to validate game functionality before release.
  • ✅ Performed database testing to ensure data integrity in MySQL and PostgreSQL.
  • ✅ Executed regression testing to maintain system stability with each update.
  • ✅ Collaborated with developers and product managers to uphold high-quality standards.

Tools & Technologies Used

Test Management: JIRA, TestRail

API Testing: Postman

Database: MySQL, PostgreSQL

Challenges & Solutions

A significant challenge was optimizing the game for lag-free gameplay as the user base grew. I worked closely with developers to identify performance bottlenecks, scale the database efficiently, and enhance overall game responsiveness across multiple devices.

Outcome & Impact

  • ✔ Successfully scaled the platform to accommodate a high user count, ensuring a seamless experience.
  • ✔ Optimized database performance to support increased traffic and maintain game stability.
  • ✔ Enhanced system performance, delivering a smooth and engaging user experience.
