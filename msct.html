Mohegan Sun CT

Company: Ipro Games

Duration: July 2022 – March 2024

Project Overview

Mohegan Sun CT is a social casino game offering an immersive experience with various casino-style games, including slots, poker, blackjack, and social game tournaments. With over 250+ games from 5+ game providers, the platform ensures engaging gameplay through in-game purchases and virtual rewards.

Role & Responsibilities

  • ✅ Conducted manual & exploratory testing to identify critical bugs.
  • ✅ Configured daily push notifications for player engagement.
  • ✅ Designed and executed test cases with documentation in JIRA.
  • ✅ Led User Acceptance Testing (UAT) before each release.
  • ✅ Performed database testing to ensure data integrity in MySQL & PostgreSQL.
  • ✅ Executed regression testing to maintain system stability.
  • ✅ Collaborated with developers & product managers for quality assurance.

Tools & Technologies Used

Test Management: JIRA, TestRail

API Testing: Postman

Database: MySQL, PostgreSQL

Challenges & Solutions

One major challenge was optimizing the game for lag-free gameplay as the user base grew. I worked closely with developers to identify performance bottlenecks, scale the database efficiently, and enhance overall game responsiveness across multiple devices.

Outcome & Impact

  • ✔ Successfully scaled the platform to accommodate a high user count.
  • ✔ Optimized database performance to support increased traffic.
  • ✔ Enhanced system performance for a smooth and engaging user experience.
